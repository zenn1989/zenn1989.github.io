Программные разработки

Программные реализации и разработки автора в открытом доступе с открытым исходным кодом

Динамика численности популяций

Реализация XSA (SimpleXSA)

Готовый набор программного кода для среды R, реализующий все этапы популяционного моделирования при помощи алгоритма eXtended survivor analysis (XSA) на базе программных продуктов FLR-project.

Реализация CMSY (SimpleCMSY)

Готовая реализация продукционной модели CMSY для ограниченных и неполных данных

Реализация площадного метода (SweptAreaBoot)

Готовая реализация площадного метода по данным научных съемок с аппроксимацией центральных тенденций через bootstrap

Экосистемный анализ

Индикаторный экосистемный анализ PCA

Готовый набор программного кода для среды R, реализующий все этапы процедуры интегрированного экосистемного анализа при помощи метода главных компонент PCA и анализа экосистемных сдвигов.

Многофакторный анализ MFA

Готовый набор программного кода для среды R, реализующий все этапы процедуры интегрированного экосистемного анализа при помощи метода многофакторного анализа MFA.

Пространственные корреляционный анализ

Набор программного кода на языке R для специалиста для выполнения пространственного корреляционного анализа с промежуточной интерполяцией Кригинга.

Прикладные программные сервисы

Программные реализации и разработки с участием автора для выполнения прикладных задач

# 1. Биостатистика

Биостатистика - закрытый проект, разрабатываемый отделом информационно-математических технологий Азово-черноморского филиала ВНИРО для сбора, хранения и анализа рыбохозяйственных данных. Проект содержит в себе 3 базы данных: ихтиологических журналов наблюдений в АЧБ, сводки судовых суточных донесений флота а так же многолетнюю статистику вылова водных живых ресурсов по отчетам АЧТУ.

Для каждой из баз данных проекта разработан интерфейс просмотра, сортировки, выборки и картирования данных а так же интерфейсы анализа.

Проект построен на базе системы управления содержимым сайта FFCMS. Для данного проекта получен патент на программную разработку в ФИПС.

# 2. Онлайн карта АзНИИРХ

Онлайн-карта АзНИИРХ - гео-информационный проект, позволяющий просматривать опубликованную информацию в режиме интерактивной карты. Проект является общедоступным и разрабатывается отделом информационно-математических технологий Азово-Черноморского филиала ВНИРО.

Используемые технологии: geoserver leaflet bootstrap

Карта позволяет получить информацию о рыболовных и рыбоводных участках АЧБ, Ростовской области и Краснодарского края; информацию о правилах любительского и промышленного рыболовства; морских режимных районах и районах опасных в навигационном плане и другие.

# 3. Оперативное прогнозирование на ИИ

FIAI - система оперативного прогнозирования промысла в Азово-Черноморском бассейне. Система предоставляет оперативный прогноз на 9 дней вперед наиболее благоприятных условий и мест вылова массовых видов водных биоресурсов для ведения эффективного промысла.

В основе прогноза лежат средства и технологии, основанные на искусственном интелекте в результате обучения системы на многолетних данных мониторинга промысла и окружающей среды.

Используемые технологии: mysql gdal python pycaret linux
