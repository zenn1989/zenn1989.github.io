Популяция

Анализ XSA

Биостатическая популяционная модель

Готовый набор программного кода для среды R, реализующий все этапы популяционного моделирования при помощи алгоритма eXtended survivor analysis (XSA) на базе программных продуктов FLR-project.

Репозиторий проекта
Экосистема

Анализ PCA

Интегрированный экосистемный анализ

Готовый набор программного кода для среды R, реализующий все этапы процедуры интегрированного экосистемного анализа при помощи метода главных компонент PCA и анализа экосистемных сдвигов.
